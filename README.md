# 🎓 Online Course Management Portal

A simple web-based application to manage courses and student enrollments using HTML, CSS, and JavaScript.

## Features

- 👤 Student Registration & Login
- 📊 Dashboard with enrolled courses count
- 📚 Course Management (Add, Delete)
- ✅ Enroll in Courses
- 📈 Course Progress Bar
- 🔍 Search Courses
- 🧩 Filter by Category
- 🎓 Certificate Page (UI)
- 💾 Data stored using LocalStorage


## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

**Project Structure**
├── index.html
├── register.html
├── login.html
├── dashboard.html
├── courses.html
├── certificate.html
├── style.css
└── script.js

## How to Run
1. Download or clone the repository
2. Open `index.html` in browser
3. Start using the application
